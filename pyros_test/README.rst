Pyros-test
==========

.. image:: https://travis-ci.org/asmodehn/pyros-test.svg?branch=indigo
    :target: https://travis-ci.org/asmodehn/pyros-test

Just a very basic ROS package, following ROS structure and recommendations.
It is useful just as a test package to validate the tricky Pyros machinery.
It contains a few test nodes that are useful for efficient ROS testing.
